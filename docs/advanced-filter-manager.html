Advanced Filter Manager

Documentation for Jira Cloud

Run on Forge

Overview

Advanced Filter Manager is a powerful Jira Cloud app that gives you full control over your saved filters. Jira's built-in filter management can feel limited when dealing with a large number of filters across multiple projects. This app provides a centralized view to organize, search, and maintain your filters effortlessly.

Key Features

  • Centralized Dashboard: View and manage all your Jira filters from a single, intuitive interface.
  • Filter Tagging: Organize and categorize your saved filters using custom tags. Our fast tag index makes it incredibly easy to manage large collections of tags.
  • Bulk Operations: Perform actions on multiple filters at once — update ownership, permissions, or delete in bulk.
  • Search & Filter: Quickly find filters by tags, name, owner, or project with our advanced search capabilities.
  • Sharing Controls: Easily manage filter sharing permissions across teams and projects.

Installation

  1. Log into your Jira instance as an administrator.
  2. Go to Apps > Find new apps.
  3. Search for "Advanced Filter Manager".
  4. Click Try it free to install.

How to Use

Once installed, the app is accessible from the Jira top navigation.

  1. Navigate to Apps > Advanced Filter Manager in the Jira top menu.
  2. The dashboard will display all filters available to you.
  3. Use the search bar to find specific filters by name or owner.
  4. Select one or more filters to perform bulk actions such as updating permissions or deleting.
